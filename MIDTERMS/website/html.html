HYPERTEXT MARKUP LANGUAGE (HTML)

HTML is used to mark up documents in the world wide web.It defines the structure and layout of a Web document by using a variety of tags and attributes. It is also an application of SGML (Standard General mark up language (ISO 8879)) and initially developed at CERN, then the IETF, then the W3C, and then also the WHATWG (Web hypertext application technology working group)

Version History:

Pre-standardization

  • HTML tags by Sir Tim Berners-Lee, late 1991
  • HTML draft by Tim Berners-Lee and Dan Cannolly published by the IETF in mid 1993

Standards

  • HTML 2.0 (RFC 18866, nov 1995)
  • HTML 3.2 (W3C Recommendation, Jan 1997)
  • HTML 4.0 (W3C Recommendation, Dec 1997)
  • HTML 4.01 (W3C Recommendation,Dec 1999)
